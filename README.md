## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AlekhyaGangopadhyay/My-beginner-projects.git
   cd My-beginner-projects
   ```

2. **Open in Jupyter/Colab:**
   ```bash
   jupyter notebook
   ```
   Or upload `.ipynb` files directly to [Google Colab](https://colab.research.google.com/).

3. **Run notebooks:**
   - `Beginner projects.ipynb` - Basic Python exercises
   - `Diabetes_Prediction.ipynb` - ML prediction model
